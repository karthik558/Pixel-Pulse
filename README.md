## Features

- **Smart URL Matching** – Target specific websites with flexible pattern matching
- **Multiple Activity Types** – Mouse movement, scrolling, network pings, or page refresh
- **Customizable Intervals** – Set execution frequency from 1 minute to hours
- **Dark/Light Mode** – Beautiful macOS-style interface that adapts to your preference
- **Privacy First** – All processing happens locally, no data collection

## Quick Start

1. **Install the Extension**
   - Download the latest release ZIP from [Releases](https://github.com/karthik558/Pixel-Pulse/releases)
   - Extract the ZIP file
   - Open Chrome and navigate to `chrome://extensions`
   - Enable "Developer mode"
   - Click "Load unpacked" and select the extracted folder

2. **Create Your First Rule**
   - Click the Pixel Pulse icon in your toolbar
   - Add a rule with:
     - **Name**: Descriptive label (e.g., "Keep Zoom Active")
     - **URL Pattern**: Website to target (e.g., `zoom.us`)
     - **Interval**: How often to pulse (in minutes)
     - **Activity**: Choose your preferred method

3. **Let It Run**
   - The extension works automatically in the background
   - View execution status in real-time
   - Use "Run Now" to test immediately

## Activity Types

| Type | Description | Best For |
|------|-------------|----------|
| **Mouse Pulse** | Simulates mouse movement and clicks | General idle prevention |
| **Scroll Nudge** | Gentle page scrolling | Reading/viewing sessions |
| **Network Ping** | Sends lightweight network requests | Keeping connections alive |
| **Page Refresh** | Reloads the page | Auto-updating dashboards |
